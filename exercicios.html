Exercícios de revisão



1) Qual é o valor do discriminante (Δ) de uma equação de segundo grau do tipo ax² + bx + c = 0?


a) Δ = b² - 4ac
b) Δ = 2ab - c²
c) Δ = a² - 4bc
d) Δ = b² - 2ac


2) Qual é a fórmula para encontrar as raízes de uma equação de segundo grau?


a) x = (-b ± √Δ) / (a - b)
b) x = (-b ± √Δ) / (2c)
c) x = (-b ± √Δ) / (a + c)
d) x = (-b ± √Δ) / (2a)



3) Se o discriminante (Δ) for igual a zero, quantas raízes reais a equação possui?


a) Infinitas raízes reais
b) 0 raízes reais
c) 2 raízes reais
d) 1 raiz real



4) Se o discriminante (Δ) for menor que zero, quantas raízes reais a equação possui ?


a) Infinitas raízes reais
b)1 raiz real
c) 0 raízes reais
d)raízes reais



5) Se o discriminante (Δ) for maior que zero, quantas raízes reais a equação possui ?


a)0 raízes reais
b)2 raízes reais
c)Infinitas raízes reais
d)1 raiz real



6) Qual é o termo responsável pela inclinação da curva de uma equação de segundo grau ?


a)Coeficiente a
b)Coeficiente b
c)Coeficiente c
d)Coeficiente d



7) Se o coeficiente a for igual a zero, a equação ainda é considerada uma equação de segundo grau ?


a)Não há coeficiente a
b)Talvez
c)Sim
d)Não



8) Qual é o número máximo de raízes reais que uma equação de segundo grau pode ter?


a)1 raiz real
b)Infinitas raízes reais
c)2 raízes reais
d)0 raízes reais



9) Qual é o nome dado às raízes iguais em uma equação de segundo grau?


a)Raízes imaginárias
b)Raízes coincidentes ou raiz dupla
c)Raízes complexas
d)Raízes distintas



10) Qual é o nome dado às raízes diferentes em uma equação de segundo grau?


a)Raízes coincidentes ou raiz dupla
b)Raízes imaginárias
c)Raízes distintas
d)Raízes complexas
